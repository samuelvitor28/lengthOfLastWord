# Length of Last Word

![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=java&logoColor=white)

## Problema

Dada uma string `s` composta por palavras e espaços, retorne o tamanho da **última palavra** da string.

- Uma palavra é definida como uma sequência de caracteres sem espaços.
- A string pode conter espaços no início ou no final.

## 🧪 Exemplos

Entrada: "Hello World"  
Saída: 5  

Entrada: "   fly me   to   the moon  "  
Saída: 4  

Entrada: "luffy is still joyboy"  
Saída: 6
